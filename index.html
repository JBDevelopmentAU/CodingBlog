My coding journey begins....

In this blog, I will take you through my coding journey, day to day, from start to finish. In each post, I will be outlining the content in which I have learnt, and the time I have spent learning code each day.

My coding journey began at the end of 2021, where I started to dive in to HTML-5 and CSS-3. The blog posts will begin as of the 1st of January, 2022. Up until this point, I have completed an Introduction to HTML-5, Intermediate HTML-5, Introduction to CSS-3 and Intermediate CSS-3. I have also touched base on Version Control using GIT, and started to become fimilar with Terminal usage using GIT Bash.
